# Extensions

* [Git Lens](https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens) More git power!
* [Git Graph](https://marketplace.visualstudio.com/items?itemName=mhutchie.git-graph) Nice SourceTree-like git graphs
* [TODO Tree](https://marketplace.visualstudio.com/items?itemName=Gruntfuggly.todo-tree) Scans code for TODO, FIXME etc and presents in the sidebar
* [Better TOML](https://marketplace.visualstudio.com/items?itemName=bungcip.better-toml) For editing TOML files such as `Cargo.toml`
* [RLS](https://marketplace.visualstudio.com/items?itemName=rust-lang.rust) The Rust language server
* [Code LLDB](https://marketplace.visualstudio.com/items?itemName=vadimcn.vscode-lldb) For debugging Rust. See Windows caveats.
* [C# by Omnisharp](https://marketplace.visualstudio.com/items?itemName=ms-vscode.csharp)


# Debugging in Rust

Install the above tools (RLS, LLDB) then [follow this guidance](https://www.forrestthewoods.com/blog/how-to-debug-rust-with-visual-studio-code/)
